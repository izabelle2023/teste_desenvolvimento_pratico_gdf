# Sistema Rede SUAS – Front-end

Este projeto consiste no desenvolvimento do front-end de uma aplicação web utilizando React JS, seguindo o padrão SPA (Single Page Application) e utilizando o Template Base do Gov.br.

## Tecnologias Utilizadas
- React JS
- React Router DOM
- Gov.br Design System

## Estrutura da Aplicação
A aplicação possui três rotas principais:
- **Acesso**: página de login com formulário de e-mail e senha.
- **Registro**: página de cadastro de usuário.
- **Painel**: área interna do sistema.

Todas as páginas são renderizadas dentro de um layout único, contendo cabeçalho, menu lateral e rodapé, conforme o Template Base do Gov.br.

## Observações Técnicas
- A autenticação é simulada no front-end, conforme escopo do teste.
- O projeto foi estruturado visando fácil integração futura com um backend.
- A navegação ocorre sem recarregamento de página (SPA).
